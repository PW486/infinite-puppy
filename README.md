# 🐶 InfinitePuppy

Explore an infinite world of adorable puppies and discover pure joy in every scroll.

## 🚀 Features
- **Endless Content**: Infinite scroll with interleaved photos and videos.
- **Modern UI**: Fluid masonry layout with a sleek lightbox modal.
- **Optimized**: Full SEO configuration (JSON-LD, Sitemap, Meta Tags).
- **Interactive**: Hover previews for videos and smooth Framer Motion animations.

## 🛠 Tech
Next.js 15, TypeScript, Framer Motion, Lucide, Pexels API.

## 🏃 Quick Start
1. Add `NEXT_PUBLIC_PEXELS_API_KEY` to `.env.local`.
2. Run `npm install && npm run dev`.
